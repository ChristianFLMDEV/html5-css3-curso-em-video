Principais formatações

Negrito/Destaque

Nesta frase temos uma palavra em negrito sem sentido semântico utilizando a tag b

Nesta outra frase temos uma outra palavra em destaque com sentido semântico utilizando a tag strong

Itálico/Ênfase

Nesta frase temos uma palavra em Itálico sem semântica

Nesta outra frase temos a mesma palavra usando o método EM para dar significado

Texto marcado

Podemos criar também um texto marcado utilizando a tag MARK.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não deve ser considerado (utilizando a tag del)

Texto inserido

Podemos marcar um texto como inserido (com a tag ins) para dar ênfase e indicar que ele foi adicionado depois

Texto sobrescrito

Para inserir coisas do tipo x2 - 3x = 0 (utilizando a tag sup)

Texto subscrito

Para inserir coisas do tipo H2O (utilizando a tag sub)
